Create a webhook for GitHub using ngrok. Step 1: Prerequisites Installed software: Git: git --version Python / Flask: python3 --version ngrok: downloaded and unzipped Verified ngrok account and obtained an authtoken. Step 2: Setup Local Flask Server Create a project folder and navigate into it: mkdir webhook-demo cd webhook-demo Create a Python virtual environment and activate it: python3 -m venv venv source venv/bin/activate # For Linux/macOS # On Windows: venv\Scripts\activate Install Flask: pip install flask Create app.py with the following content: from flask import Flask, request, jsonify app = Flask(__name__) @app.route("/webhook", methods=["POST"]) def webhook(): data = request.json print("Webhook received:", data) return jsonify({"status": "success"}), 200 if __name__ == "__main__": app.run(port=5000) Run the Flask server: python app.py Expected Output: * Running on http://127.0.0.1:5000/ Step 3: Setup ngrok Tunnel Authenticate ngrok with your token: ./ngrok authtoken Start a tunnel to the Flask server port 5000: ./ngrok http 5000 Output will show a public URL, e.g.: Forwarding https://abcd1234.ngrok.io -> http://127.0.0.1:5000 This public URL will be used in GitHub webhook. Step 4: Configure GitHub Webhook Open your GitHub repository. Go to: Settings ‚Üí Webhooks ‚Üí Add webhook Enter details: Payload URL: https://abcd1234.ngrok.io/webhook Content type: application/json Secret: (Optional) Events: Select Just the push event or all events as required Click Add webhook. Step 5: Test Webhook Make a commit/push in the repository. Flask server console should show: Webhook received: {...json payload...} This confirms webhook is working via ngrok tunnel. ________________________________________
